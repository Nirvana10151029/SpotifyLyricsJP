# Changelog

## 2.0.10 — 2026-07-22

- Remaster・Live・Acoustic・feat.・全角記号などを除去した複数の検索候補を生成し、歌詞の検索範囲を拡大
- 複数アーティスト表記を分解し、曲名・アーティスト・アルバム・再生時間を重み付きで評価
- 明らかな別曲、短い誤候補、カラオケ・Instrumental候補を除外する安全基準を強化
- Spotify Playerの追加メタデータ形式に対応し、再生曲を認識できる範囲を拡大
- LRCLIBとLyricaを並列検索し、安全に一致した同期歌詞を優先することで取得を高速化
- 検索タイムアウトとキャッシュ件数を調整

## 2.0.6 — 2026-07-20

- 互換歌詞パネルがSpotify本来の右側パネルを覆い、アーティスト画像・アルバム情報・Canvasが見えなくなる問題を修正
- Spotifyの右側パネルを自動検出し、歌詞パネルをその左隣へ配置
- ウィンドウ幅やSpotify側パネルの開閉・サイズ変更に追従して位置を再調整
- 並列表示できない狭い画面では従来の重ね表示へ自動フォールバック

## 2.0.5 — 2026-07-20

- 互換パネルで現在行が変わるたびに歌詞全体を再描画し、スクロールが先頭からやり直される問題を修正
- 再生中は歌詞DOMを維持し、現在行のハイライトだけを更新するよう変更
- パネル外ではなく歌詞欄内部だけを滑らかにスクロールし、現在行を中央付近へ維持

## 2.0.4 — 2026-07-20

- Spotify更新により `Spicetify.Panel` または `Spicetify.Topbar` が提供されない環境に対応
- 標準の右パネルAPIが使えない場合、Spotify内へ直接表示する互換パネルと緑色の開閉ボタンへ自動切り替え
- `Spicetify.PopupModal` が使えない場合にもAPIキーを設定できる互換ダイアログを追加
- 右パネルAPI不足を起動失敗として扱わないよう修正

## 2.0.3 — 2026-07-20

- `expose_apis=1` の設定後に通常の `apply` だけが成功し、右パネルAPIが実際には組み込まれない問題を修正
- 既存バックアップを復元してから必ず再前処理し、初回環境ではバックアップ作成へ自動フォールバックするよう変更
- 必要なSpicetify APIが読み込まれない場合、待機し続けず30秒後に再インストール案内を表示

## 2.0.2 — 2026-07-20

- 既存のSpicetifyバックアップがある環境で `backup apply` が停止する問題を修正
- 通常の `apply`、初回用 `backup apply`、既存バックアップ復旧用 `restore backup apply` の順で自動復旧
- 拡張機能が利用するSpicetify APIを明示的に有効化

## 2.0.1 — 2026-07-20

- Windowsのコマンド画面で起動用BATが壊れて読まれる問題を修正
- Windows向けのCRLF改行・PowerShell 5.1互換UTF-8 BOMで梱包
- 起動するファイルを `START-INSTALL.bat` と `START-UNINSTALL.bat` に整理し、直接開く必要のないPowerShell本体と区別
- セットアップ画面を成功・失敗どちらの場合も閉じずに確認できるよう修正

## 2.0.0 — 2026-07-20

- Spotify公式サイト版とSpicetifyに対応し、別ウィンドウからSpotify右側の内蔵パネルへ変更
- 再生曲の自動検出、同期ハイライト、自動スクロールをSpicetify Player APIへ移植
- LRCLIBで通常歌詞しか見つからない場合も検索を続け、別候補またはLyricaの同期歌詞を自動優先
- LRCLIB、Lyrica経由のYouTube Music・NetEase・Megalobiz・Musixmatch・SimpMusic、Lyrics.ovhに対応
- 曲名・アーティスト・アルバム・再生時間の照合と、45秒以上異なる候補の除外を移植
- 「再取得」と「別ソース」をSpotify内蔵パネルへ追加
- 取得失敗後は自動再試行せず、曲変更または手動操作まで停止するよう実装
- 無料翻訳、Gemini、DeepL、GPTの切り替えとAPI設定画面を移植
- Gemini・GPTは行ID付きJSONを検証し、歌詞行の結合・省略による同期ずれを防止
- DeepL API Free/Proのエンドポイントをキーから自動判定
- GPT自然訳をGPT-5.6 SolとResponses APIの構造化出力に更新
- AI翻訳失敗時の無料翻訳フォールバックを実装
- Web版Spotify・Spicetify・拡張機能を確認して導入するWindows用インストーラーを追加
- 拡張機能だけを安全に取り外すアンインストーラーを追加
